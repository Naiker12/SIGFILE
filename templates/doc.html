¿Qué es SIG File?

SIG File es un sistema de gestión de archivos territoriales para la Alcaldía de Cartagena. Convierte archivos complejos (.zip, .ram, .xlsx) en Excel organizados automáticamente.

1

Subir Archivo

Carga tu archivo desde tu computadora

Procesamiento

El sistema organiza los datos automáticamente

3

Descargar

Obtén un Excel limpio y organizado

Procesamiento Excel

Extrae columnas específicas de archivos Excel grandes

  • Selección de columnas
  • Múltiples hojas
  • Filtrado automático

Extracción ZIP

Genera inventarios automáticos de archivos comprimidos

  • Nombres de archivos
  • Tamaños y fechas
  • Estructura de carpetas

Archivos RAM

Procesa datos territoriales especializados

  • Datos geográficos
  • Coordenadas exactas
  • Referencias catastrales

Excel (.xlsx)

Lectura de múltiples hojas, selección de columnas específicas y exportación optimizada.

ZIP (.zip)

Extracción sin descomprimir, análisis de estructura y generación de inventarios.

RAM (.ram)

Lectura de datos geográficos, extracción de coordenadas y conversión tabular.

Instrucciones de Uso

  1. Subir: Selecciona tu archivo (.xlsx, .zip, .ram)
  2. Procesar: El sistema analiza automáticamente (5-30 segundos)
  3. Descargar: Obtén tu Excel organizado

Ejemplo Excel

Excel con 500 propiedades → Solo columnas necesarias en segundos

Ejemplo ZIP

ZIP con 200 archivos → Inventario completo automático
