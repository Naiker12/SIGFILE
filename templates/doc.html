¿Qué es SIG File?

SIG File es un sistema innovador de gestión de archivos territoriales diseñado específicamente para la Alcaldía de Cartagena.

Objetivo principal: Simplificar el trabajo manual y optimizar el procesamiento de grandes volúmenes de datos territoriales.

Propósito Principal

Convertir archivos complejos (.zip, .ram, .xlsx) en Excel resumidos y organizados.

Elimina la necesidad de copiar nombres y datos uno por uno manualmente, ahorrando horas de trabajo.

Flujo de Trabajo

1

Subir Archivo

Cargas tu archivo (.zip, .ram, .xlsx) desde tu computadora de forma segura.

2

Procesamiento

El sistema extrae y organiza los datos automáticamente en segundos.

3

Descargar

Obtienes un Excel limpio, organizado y listo para usar.

Procesamiento Excel

Selecciona específicamente las columnas que necesitas de archivos Excel grandes.

  • Filtrado automático de datos
  • Múltiples hojas de cálculo
  • Exportación optimizada

Extracción ZIP

1

Subir Archivo

Selecciona el archivo que deseas procesar desde tu computadora. El sistema acepta archivos .xlsx, .zip y .ram.

💡 Tip: Asegúrate de que el archivo no esté corrupto y sea menor a 100MB.
2

Procesamiento

El sistema analiza automáticamente el archivo y extrae la información relevante según el tipo.

⏱️ Tiempo: El procesamiento toma entre 5-30 segundos dependiendo del tamaño.
3

Descargar

Una vez procesado, podrás descargar el archivo Excel resultante con los datos organizados.

✅ Resultado: Excel optimizado con solo la información que necesitas.

Ejemplo Excel

Problema: Tienes un Excel con 500 propiedades y 30 columnas, pero solo necesitas "Código Catastral" y "Dirección".

Solución: SIG File extrae automáticamente solo esas 2 columnas, creando un archivo limpio y manejable en segundos.

Ejemplo ZIP

Problema: Tienes un ZIP con 200 planos y necesitas un inventario completo de todos los archivos.

Solución: SIG File genera automáticamente un Excel con nombres, tamaños, fechas y tipos de todos los archivos contenidos.
